# KOVARI

A social travel platform designed for smart group matching and verified connections.

## Overview
KOVARI is a sophisticated travel platform engineered to connect individuals through AI-driven compatibility and shared travel itineraries. The system facilitates the creation of trusted travel circles, ensuring reliable and secure group travel experiences for verified participants.

## Core Features
- Smart travel matching
- AI-based compatibility
- Travel circles (small groups)
- Shared itinerary planning
- Verified user system
- Trust and safety mechanisms

## System Overview
The platform leverages a modern, scalable technology stack:
- **Frontend**: Next.js, TypeScript, Tailwind CSS, ShadCN UI
- **Backend**: Supabase (Auth, Database, Storage), Edge Functions
- **Infrastructure**: Vercel, Redis
